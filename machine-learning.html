novel ai diffusion

this won't be a guide on how to use the novelai diffusion model. there are already plenty of guides online that would do a far greater job at that than i ever could. this is mainly just to document my findings.

be careful when downloading checkpoints, it's possible for them to allow for arbitrary code execution. you wouldn't download a file called sex.exe, and you shouldn't download a file named sex.ckpt

i'm using automatic1111's webgui (specifically, pulled from commit 7432b6f / 7432b6f4d2c3001895fc75411a34afae1810c1a2, which is the most up to date version at the time of testing)

these experiments are being run on a system using a GTX1050 mobile 3GB. This is important to note, as the 1050 series of cards are reported to have significantly different outputs than almost every other card for... seemingly no reason. (see: https://imgur.com/a/DCYJCSX, 'are you using a 1050ti?') running with the exact settings used for the asuka test does indeed give me an incorrect result, so once again, keep this in mind and take my results with a grain of salt.

test 1: xformers

this is to test how much (or even if) xformers helps with speeding up the outputs

NOTE: this data is from a 1050, using a 3050 or above would result in times like 1/10th as long

Seed Average time (xformers on) Average time (xformers off)
228922 1m, 41s 2m, 02s
842459461 1m, 41s 1m, 59s
610670453 1m, 41s 1m, 59s

xformers did generally save quiet a bit of time and had almost no impact on the outputs.

i say almost because it does cause some consistancy issues with the background and accessories. notice the chair in the first one, or his skirt in the second and third. however, this is nearly impossible to notice unless you are specifically trying to get the same background multiple times. the faces and stuff remain perfectly identicle as far as i can tell.

it might also just be another issue caused by 1050s? i don't have another gpu to test on so i can't really confirm that though.

overall i'd say its worth using the --xformers flag if you are going to be doing a lot of generating. xformers doesn't take particually long to build either so i'd say it's worth building it yourself if you can't find a .whl for your gpus cuda architecture (it only took like 15 minutes to build for my gpu)

kinda important update: apparently xformers can cause heavy degredation. i personally haven't noticed this (maybe its whats causing my results with xformers to differ slightly?), but it is seemingly a very legitimate issue (see: here.) maybe hold off on using xformers unless you are on a low end card
